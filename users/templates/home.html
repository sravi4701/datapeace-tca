Some notes

  • I am using Django and Django rest Framework to build this API

  • I'm using sqlite3 database

  • For running the API, Django rest framework has it's own browsable api console

    If you want to run it on terminal then use curl Something like: curl -H "Content-type:application/json" http://127.0.0.1:8000/api/users/ | json_pp

  • Use PUT http method for fully update the info, in this you have to provide all the fields and it's vavlues

    Use PATCH http method for partially update the info, in this you have to provide some of the fields and it's vavlues

App Explanation

  • First I have created a django project datapeace using django-admin

  • Then I created a separate app users using python manage.py

  • Then I have created an api package inside users app for all business logic of api
